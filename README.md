# Welcome to skript.help!
skript.help is (hopefully) the best resource available for getting tutorials on Skript. We haven't a lot of articles because we're so new, but you can change that! Our entire website, including the source code, and all of our articles, are available on [GitHub](https://github.com/honkling/skript.help)!

# How do I contribute?
You can contribute to skript.help by going to our [GitHub](https://github.com/honkling/skript.help/tree/pages), and forking the repository, by clicking the **Fork** button at the top right of the page.
After you've forked the repository, you can go to one of the several categorizing folders (ex. `features`, `tutorials`, `examples`), and create a new file with a short name for your article, and make sure to add `.mdx` to the end of the name.
For example, the file name could be `commands.mdx`, `functions.mdx`, etc. If you need to use spaces in the name, use a dash to represent it, for example, `good-practices.mdx`.

We recommend you familarize yourself with the Markdown language before contributing.
